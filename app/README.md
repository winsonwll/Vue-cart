
## 本目录结构如下
	|- subject #活动
	    |- project          # 项目名
	        |- html        # 静态HTML资源文件
	        |- dist         # 产出目录
	            |- css
	            |- js
	        |- src          # 开发目录
                |- js
                |- scss  
	        |- appconfig.json  #配置文件 必须存在
            |- readme.txt   # 说明文件
                |- project  # 项目描述
                |- author   # 开发者
                |- date     # 创建时间
